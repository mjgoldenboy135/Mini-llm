04 — How It Works
1. Token + Positional Embeddings — Each character → D-dim learned vector. Position index → separate D-dim vector. Sum them.
2. Multi-Head Causal Self-Attention — Project to Q,K,V. Scaled dot-product scores. Triangular causal mask. Softmax → weighted V. Concat heads → Wo.
3. Residual + LayerNorm — Pre-norm: x = x + block(LN(x)). Full LN backward (not approximated).
4. Feed-Forward Network — D → F → D per position with ReLU. Correct backward: dBf = dpre @ Wf1.T.
5. Cross-Entropy over all S positions — S× more gradient signal per forward pass than single-token prediction.
6. Grad Clip + Adam — Global norm ≤1.0; Adam β₁=0.9, β₂=0.999 with bias correction.
7. LR Warmup + Cosine Decay — Linear ramp 8% of steps; cosine 3e−3 → 5e−5.
8. Auto-Train Sources — Wikipedia, HackerNews, JokeAPI, AdviceSlip, On-This-Day. All CORS-enabled, no API keys. Fetched every 2 min, appended to a 200KB rolling corpus.
9. Persistent Memory — Weights + Adam states saved to localStorage every 100 steps, on Stop, and on page refresh/close. Next visit: knowledge restored automatically. Reset only wipes on explicit confirmation.
