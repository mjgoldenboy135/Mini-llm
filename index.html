04 — How It Works
1. Token + Positional Embeddings — Each character → D-dim learned vector. Position index → separate D-dim vector. Sum them.
2. Multi-Head Causal Self-Attention — Project to Q,K,V. Compute scaled dot-product scores. Apply triangular mask so position i only sees ≤i. Softmax → weighted sum of V. Concat heads → Wo.
3. Residual Connections — x = x + attn(LN(x)) and x = x + ffn(LN(x)). Prevents vanishing gradients across layers.
4. LayerNorm — Normalize each position vector: ŷ = γ⋅(x−μ)/σ + β. Full backward formula implemented (not approximated).
5. Feed-Forward Network — Two linear layers with ReLU: D → F → D per position. Independent across positions.
6. Cross-Entropy Loss — Averaged over all seq positions simultaneously—gives S× more gradient signal per forward pass vs single-token prediction.
7. Full Manual Backprop — Chain rule through softmax, attention, LN, residuals, FFN—zero autograd libraries.
8. Gradient Clipping + Adam — Global norm ≤1.0. Adam β₁=0.9, β₂=0.999 with bias correction applied to every parameter.
9. LR Warmup + Cosine Decay — Linear ramp first 8% of steps; cosine anneal 3e−3 → 5e−5.
10. Nucleus Sampling — Top-p keeps the minimal token set covering probability mass p.
