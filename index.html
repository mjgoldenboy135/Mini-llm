04 — Under the Hood
1. Character Embeddings — Each token gets a trainable dense vector (embed_dim). Lookup + concatenate context window → [C × E] input vector.
2. MLP Layer 1 — Linear [C×E → H] + ReLU activation. He initialisation for stable early gradients.
3. MLP Layer 2 — Linear [H → V]. Raw logits over vocabulary.
4. Cross-Entropy Loss — −log P(target | context). Softmax computed in log-space for numerical stability.
5. Backpropagation — Full manual backprop through all layers including embedding scatter-add.
6. Gradient Clipping — Global norm clipped to 1.0 to prevent exploding gradients.
7. Adam Optimiser — β₁=0.9, β₂=0.999. Adaptive per-parameter learning rates with bias correction.
8. LR Warmup + Cosine Decay — Linear warmup for 8% of steps, then cosine anneal from 3e-3 → 5e-5.
9. Train / Val Split — 90% train, 10% val. Both curves plotted live to monitor overfitting.
10. Nucleus Sampling — Top-p keeps the minimal token set covering probability mass p, then re-normalises and samples.
